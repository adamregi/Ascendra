# Feature-to-Skill Map — Ascendra

> **Purpose**: Maps task categories to relevant Skills/ folder searches. AI agents use this to automatically discover which skills to load before implementing a feature.

---

## How to Use This Map

1. Identify which feature/task category your current work falls under
2. Search the `Skills/skills/` directory for the listed skill names
3. Read the matching `SKILL.md` files
4. Combine guidance from multiple skills when applicable
5. Apply alongside project architecture (AGENTS.md takes precedence)

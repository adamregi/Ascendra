# Skill Categories — Ascendra

> **Purpose**: A categorized index to efficiently navigate the 24,000+ skills in the `Skills/` directory.

---

## 1. Discovery Workflow

Do not attempt to read or list all skills. When starting a task, follow this workflow:

```text
Category
   ↓
Search (using keywords below)
   ↓
Identify Specific Skill (.md file)
   ↓
Load and Read
```

## 2. Categories & Search Keywords

### 📱 Flutter Architecture
- **Keywords**: `flutter`, `clean-architecture`, `repository-pattern`, `dto`, `freezed`, `json-serializable`.
- **Use when**: Setting up a new feature folder, defining models, or mapping backend data to the client.

### 🌊 State Management (Riverpod)
- **Keywords**: `riverpod`, `riverpod-generator`, `async-value`, `state-notifier`, `cacheFor`, `provider-lifecycle`.
- **Use when**: Fetching data, managing local state, or handling loading/error UI states.

### 🎨 UI & Responsive Design
- **Keywords**: `responsive`, `ui`, `layout`, `design-system`, `mobile-first`, `tablet`, `forms`, `validation`.
- **Use when**: Translating reference screens into Flutter code, ensuring proper layout across devices, or building forms.

### 🐘 Database & Backend (PostgreSQL)
- **Keywords**: `postgresql`, `rpc`, `plpgsql`, `rls`, `row-level-security`, `migrations`, `triggers`, `jsonb`, `materialized-views`.
- **Use when**: Writing migrations, modifying schema, creating Backend-for-Frontend RPCs, or optimizing queries.

### ⚡ Supabase Core
- **Keywords**: `supabase-auth`, `supabase-storage`, `edge-functions`, `realtime`, `webhooks`.
- **Use when**: Handling user authentication, uploading files (proofs/avatars), or listening to database streams.

### 🧠 AI & Advanced Capabilities
- **Keywords**: `pgvector`, `rag`, `embeddings`, `gemini`, `llm`, `ltree`, `hierarchical-data`.
- **Use when**: Working on the AI Assistant, document indexing, or querying the MLM network tree.

### 🧪 Quality & Testing
- **Keywords**: `widget-test`, `unit-test`, `integration-test`, `mockito`, `pgtap`, `performance-profiling`.
- **Use when**: Writing tests, fixing bugs, or optimizing app performance.

### 🚀 Deployment & DevOps
